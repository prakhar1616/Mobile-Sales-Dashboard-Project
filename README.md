# 📊 Mobile Sales Dashboard

✨ An interactive Power BI Dashboard that transforms raw mobile sales data into powerful insights for smarter decision-making.

# 🚀 Overview

This dashboard provides a complete analysis of mobile sales across different dimensions like city, brand, time, and payment methods. It helps understand trends, customer behavior, and product performance in a simple and visual way.

# 🔥 Key Highlights

## 📈 Total Sales Insights

💰 Revenue: 769M

🧾 Transactions: 4K

📦 Quantity Sold: 19K

## 🗺️ City-wise Analysis

Visual map showing sales distribution across major cities

## 📅 Monthly Trends

Track performance and sales patterns throughout the year

## ⭐ Customer Ratings

Breakdown into Good, Average, Poor feedback

## 💳 Payment Methods
UPI | Debit Card | Credit Card | Cash

## 📱 Top Mobile Models

Identify best-selling devices

## 📊 Brand Performance

Compare brands based on sales and transactions

## 📆 Day-wise Trends

Discover peak sales days
